# Docker Basics Practice

This folder contains my Docker hands-on practice while learning Docker.

Topics covered:
- docker run
- docker ps
- docker stop
- docker start
- docker rm
- docker rmi
- docker pull
- docker exec
- port mapping
- container lifecycle

Goal:
Build strong Docker fundamentals before moving to Docker Compose and Kubernetes.
